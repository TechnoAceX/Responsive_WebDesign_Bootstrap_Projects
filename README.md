# Responsive Web Design & Bootstrap Projects

A collection of responsive web development projects built with **HTML5**, **CSS3**, **Bootstrap 5**, and **JavaScript**. These projects demonstrate responsive layouts, Bootstrap Grid, dashboard design, and interactive UI components.

---

## 📂 Projects

### 1. Responsive Landing Page (`index1.html`)

A modern landing page built using pure HTML and CSS.

**Features**
- Responsive header and navigation
- Hero section
- Responsive card layout
- Information section
- Responsive footer
- CSS Grid & Flexbox
- Media Queries

---

### 2. Bootstrap 5 Grid System (`index2.html`)

A complete demonstration of Bootstrap's responsive grid system.

**Features**
- Containers
- Rows & Columns
- Responsive Breakpoints
- Nested Grid
- Column Offsets
- Column Ordering
- Gutters
- Alignment Utilities
- Responsive Layout Examples

---

### 3. Responsive Dashboard Layout (`index3.html`)

A responsive dashboard showcasing Bootstrap Grid with custom media queries.

**Features**
- Responsive navigation bar
- Sidebar layout
- Dashboard cards
- Statistics section
- Article cards
- Sticky sidebar
- Responsive footer
- Custom Media Queries

---

### 4. Bootstrap Interactive UI (`index4.html`)

A showcase of Bootstrap interactive components with JavaScript.

**Features**
- Alerts
- Buttons
- Modal Popups
- Toast Notifications
- Progress Bar
- Interactive Counter
- Form Modal
- Responsive Design

---
